# KasTaruna17 🇮🇩

Sistem pencatatan kas Karang Taruna RT 05 untuk persiapan kegiatan HUT RI 17 Agustus 2025.

## Fitur Utama

- **Dashboard Keuangan**: Ringkasan saldo, pemasukan, pengeluaran, iuran warga, dan visualisasi progress target dana.
- **Pencatatan Pemasukan & Pengeluaran**: Manajemen transaksi keuangan dengan kategori dinamis.
- **Iuran Warga**: Monitoring status pembayaran iuran warga (lunas/belum lunas).
- **Laporan Keuangan**: Laporan terperinci yang siap cetak (printer/PDF) dan dapat dibagikan melalui WhatsApp.
- **Persistensi Data**: Penyimpanan otomatis menggunakan browser Local Storage agar data tidak hilang setelah refresh.
